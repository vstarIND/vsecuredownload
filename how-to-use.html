How to Install V Secure [For Students]

  1. Download the V Secure ZIP file from the Download Page.
  2. Extract the ZIP file to a folder on your computer.
  3. Open Google Chrome and go to chrome://extensions/.
  4. Turn on Developer mode (toggle switch on the top right).
  5. Click on Load unpacked.
  6. Select the folder where you extracted the V Secure extension files.
  7. V Secure will now appear in your Chrome extensions list and is ready to use!
